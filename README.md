# CourseWork
Курсовой проект по модулю «Автоматизация тестирования» для профессии «Инженер по тестированию»

# **Автоматизация тестирования комплексного сервиса, взаимодействующего с СУБД и API банка**

## Начало работы

Чтобы клонировать репозиторий для дальнейшего запуска на локальном ПК, необходимо:
* зайти на GitHub на страницу данного репозитория
* Нажать на кнопку code, затем скопировать URL ссылку 
* в командной строке ввести команду "git clone" + скопированный URL


               ИЛИ
* в командной строке ввести команду: "git clone https://github.com/A-Serafimskaya/CourseWork.git"

### Prerequisites
Что должно быть установлено на ПК:
* Git
* Intellij IDEA
* Браузер
* Docker
* DBeaver

### Запуск БД, jar- приложения и тестов


* Командой **docker compose up** запустить контейнер

* Командой **java -jar artifacts/aqa-shop.jar** Запустить приложение aqa-shop.jar 

* Командой **./gradlew clean test** запустить тесты
