# Отчет о проведенной автоматизации
## 1. Было запланировано 
* Позитивные и негативные UI-тесты веб-интерфейса приложения для двух опций: оплата по карте, оплата в кредит
* Проверка корректности внесения информации приложением в базу данных (запросы в базу) об успешности операции (APPROVED/DECLINED)
* Формирование репортов  Allure 
* Подготовка issues
* Подготовка отчетной документации о проведенном тестировании и автоматизации

## 2. Реализовано
Для каждого из вариантов оплаты:
* реализовано по 28 тестов: 23 успешных и 5 неуспешных
* подготовлено 5 issue для оплате по карте
* сформирован отчет Allure

## 3. Причины, по которым что-то не было реализовано
Не реализованы issue для опции покупки в кредит, т.к. они аналогичны покупке по карте
